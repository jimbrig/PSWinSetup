# Install-ChocoHelpers

## SYNOPSIS
Installs initial Chocolatey helper packages.

## SYNTAX

```
Install-ChocoHelpers [<CommonParameters>]
```

## DESCRIPTION
Installs the following:
 - boxstarter
 - choco-cleaner
 - choco-package-list-backup
 - instchoco
 - chocolateygui
 - 7zip.install
