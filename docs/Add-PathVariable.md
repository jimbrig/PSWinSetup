# Add-PathVariable

## SYNOPSIS
Adds a specified value to System %PATH% variable.

## SYNTAX

```
Add-PathVariable [[-addPath] <String>] [<CommonParameters>]
```

## DESCRIPTION
Adds a specified value to System %PATH% variable.

## EXAMPLES

### EXAMPLE 1
```
Add-PathVariable "C:\bin"
```
